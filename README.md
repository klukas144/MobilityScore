# MobilityScore
Android App to calculate the Mobility Score of your current position (under development)
This app is a student group project for "Digitalisierung und Mobilsoftware" in the Bachelor's degree "Verkehrssystemmanagement" at Hochschule Karlsruhe, Germany.
We have to develop an app that provides a Mobility Score based on the users current location including data as the nearest stations or nearest available shared bikes.
